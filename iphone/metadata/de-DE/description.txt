Wir bei Organic Maps glauben, dass Privatsphäre ein grundlegendes Menschenrecht ist:

• Organic Maps ist ein Open-Source-Projekt, das von einer unabhängigen Gemeinschaft betrieben wird
• Der Code ist Open Source
• Die Kartendaten stammen von OpenStreetMap, einem ebenfalls von der Community betriebenen Projekt
• Wir schützen Ihre Privatsphäre vor den neugierigen Augen von Großunternehmen
• Bleiben Sie sicher, egal wo Sie sind
